Contracts Manager

Never miss a contract renewal. Track every client agreement — expiration dates, FAR/DFAR compliance requirements, perpetual contracts, and renewal windows — all in one native macOS app.

Always Know Where You Stand

Four status indicators in the toolbar give you an instant read on every contract across your entire client roster.

Expired

Instantly flags any contract that has passed its expiration date. Click to see every affected client.

No Contracts

Identifies clients with no agreements on file — so no one slips through the cracks.

Need Renewal

Alerts you when renewable contracts are approaching expiration within your configured window.

Old Perpetual

Surfaces perpetual contracts that haven't been reviewed in years — before old agreements become a liability.

Every Client. Every Contract. One View.

The sidebar gives you a searchable, color-coded list of all your recipients. Red names need renewal. Orange names have aging perpetual contracts. Green means you're in the clear.

  • Fixed-term contracts with live countdowns
  • Perpetual contracts tracked separately
  • Active, Expiring Soon, and Expired status badges
  • Right-click to edit, delete, or mark as reviewed
  • macOS notifications 45 days before expiration

Add Contracts in Seconds

A clean, focused sheet for adding new contracts. Choose the contract type, set a fixed term or mark it perpetual, pick the signing date, and flag it for renewal tracking — all without leaving the flow.

  • Custom contract type library (NDA, MSA, DFAR Flowdowns, and more)
  • Fixed term in months or years
  • Perpetual toggle for non-expiring agreements
  • Renews on expiration flag with automatic notification scheduling

Built for Government & Defense Contractors

Assign FAR, DFARS, and ITAR compliance requirements to any client for quick reference. Build your own requirements library under Manage, then assign relevant clauses per recipient.

  • Global requirements library — fully customizable
  • Per-client clause assignment
  • Numbered, text-selectable table — easy to copy
  • Filter sidebar to show only clients with requirements
  • Included in database exports and imports

Everything You Need. Nothing You Don't.

📋

Smart Filters

Filter by Expired, No Contracts, Perpetual, Need Renewal, Old Perpetual, Have Requirements, or by contract type. Search and filter work together in real time.

🔔

Menu Bar Alerts

The menu bar icon blinks whenever any alert condition is active. One click shows a summary and opens the main window — no need to keep the app in the foreground.

♻️

Old Perpetual Tracking

Set an age threshold for perpetual contracts. When one ages past it, you're flagged. Use Mark as Reviewed to reset the clock and build an audit trail.

💾

Portable Database

Export everything to a clean JSON file — recipients, contracts, requirements, review dates. Open it on any Mac running Contracts Manager and pick up exactly where you left off.

🏢

Your Branding

Replace the sidebar logo with your company's mark in one click. Stays persistent across launches.

⚡

Native macOS

Built with SwiftUI and SwiftData — fast, lightweight, and right at home on your Mac. No subscriptions, no cloud sync required.
