Frequently Asked Questions

Where is my data stored?

All data is stored locally on your Mac in ~/Library/Application Support/ContractsManager/. Nothing is sent to the cloud or any external server.

How do I back up my database?

Go to File → Save Database… (⌘S) to export a portable JSON file. You can restore it on any Mac running Contracts Manager via File → Open Database… (⌘O).

How do I add a contract type or requirement?

Use the Manage menu in the toolbar. Choose Contract List to add contract types, or Contract Requirements List to add FAR/DFAR clauses and other requirements to the global library.

What does "Mark as Reviewed" do for perpetual contracts?

Right-clicking a perpetual contract and choosing Mark as Reviewed stamps today's date on that contract. The Old Perpetual age warning then counts from the review date instead of the original signing date — effectively resetting the clock and creating an audit trail.

How do I set up the renewal and old perpetual warnings?

Go to Manage → Set Periodic Renewal Period to configure how many days before expiration you want the renewal alert. Go to Manage → Set Old Perpetual Period to set the age threshold (in years) for perpetual contract warnings. Set either to 0 to disable it.

When do notifications fire?

For any contract with Renews on Expiration enabled, Contracts Manager schedules a macOS notification 45 days before the contract expires. Make sure notification permission is granted in System Settings → Notifications → Contracts Manager.

Can I use my own company logo?

Yes. Go to Edit → Change Logo… to replace the sidebar logo with a PNG, JPG, TIFF, or GIF. The image is saved and persists across launches. Use Edit → Remove Logo to revert to the default.

What macOS version is required?

Contracts Manager requires macOS 14 Sonoma or later.
